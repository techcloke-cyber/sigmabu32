# 🤖 Jarvis - AI-Powered Discord Bot with Minecraft Plugin Compilation

[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://python.org)
[![Discord.py](https://img.shields.io/badge/discord.py-2.3.2-blue.svg)](https://github.com/Rapptz/discord.py)

## ✨ Features

- 🎫 **Advanced Ticket System** - Multi-department tickets with claim system
- 🤖 **Multi-AI Support** - DeepSeek, ChatGPT, Gemini, Claude, Llama, and 200+ models via OpenRouter
- 🔨 **Minecraft Plugin Compilation** - Generate AND compile Java plugins from natural language
- ✅ **Verification System** - Self-serve role verification
- 🛡️ **AutoMod** - Spam protection, banned words, anti-link
- 📊 **Live Counters** - Automatic member & Minecraft server status
- 🎭 **Reaction Roles** - Persistent role assignment buttons
- ⚙️ **Full Moderation** - Ban, kick, timeout, warn, lock, clear
- 🖼️ **Image Analysis** - Upload images and ask Jarvis about them
- 📁 **File Processing** - Read and analyze code files, ZIP archives
